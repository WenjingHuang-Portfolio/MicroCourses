Introduction:

The Cloud Computing Essentials course demystifies cloud technologies, focusing on the three major service providers: Amazon Web Services (AWS), Microsoft Azure, and Google Cloud Platform (GCP). Learners will gain an understanding of Infrastructure as a Service (IaaS), Platform as a Service (PaaS), and Software as a Service (SaaS) models, each offering different levels of control and functionality. The course explores key concepts such as virtualization, which allows for efficient use of physical hardware, and cloud security practices that protect data and applications in virtual environments. With 35 hours of content accessible over 2 months, participants will complete hands-on labs to gain real-world experience deploying and managing cloud resources. This course is ideal for those with basic IT knowledge who wish to gain an essential understanding of cloud services and their applications in modern business.

Course Content:

  • Cloud Computing Models
  • Cloud Service Providers
  • Virtualization
  • Security in the Cloud
  • Hands-on Labs
